# 个人待办事项管理系统

基于Vue 3的移动端H5待办事项管理系统，采用四象限法则管理待办事项。

## 功能特性

- 📱 移动端H5适配
- 🎯 四象限待办事项管理
  - 着急重要
  - 着急不重要
  - 不着急重要
  - 不着急不重要
- 📝 添加、删除待办事项
- 💾 本地存储数据
- 📊 个人中心统计信息
- 🏠 底部导航栏
- ➕ 底部居中添加按钮

## 技术栈

- Vue 3
- Vue Router 4
- Vite
- CSS Grid
- LocalStorage

## 安装与运行

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## 使用说明

1. **添加待办事项**：点击底部居中的+按钮，填写标题、描述并选择优先级
2. **删除待办事项**：点击待办事项右侧的✕按钮
3. **查看统计**：在个人中心查看各象限待办数量
4. **清空所有**：在个人中心点击清空按钮（谨慎操作）

## 优先级说明

- **第一象限（着急重要）**：紧急且重要的事情，如截止日期临近的项目
- **第二象限（着急不重要）**：紧急但不重要的事情，如临时会议
- **第三象限（不着急重要）**：不紧急但重要的事情，如学习新技能
- **第四象限（不着急不重要）**：不紧急也不重要的事情，如刷社交媒体

## 数据存储

待办事项数据存储在浏览器的LocalStorage中，刷新页面后数据不会丢失。
